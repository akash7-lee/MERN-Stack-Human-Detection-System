### Installation

1. **Install Backend Dependencies**
```bash
cd backend
npm install
```

2. **Install Frontend Dependencies**
```bash
cd ../frontend
npm install
```

### Running the Application

1. **Start MongoDB** (if not running)
```bash
mongod
```

2. **Start Backend Server**
```bash
cd backend
npm run dev
```

3. **Start Frontend** (in new terminal)
```bash
cd frontend
npm start
```

4. **Open Browser**
Navigate to: http://localhost:3000
