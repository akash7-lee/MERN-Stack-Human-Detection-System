 MERN Stack Human Detection System
A real-time AI-powered human detection web application built with the MERN stack and TensorFlow.js. This system can detect people in images and live webcam feeds using advanced machine learning models.

 Fast Performance - Optimized for real-time processing
 Privacy Focused - All processing happens in your browser
